    1. Introdução

    O Rich Picture trata-se de uma forma de representar atividades, fluxos e ideias para um processo através de um desenho. Não existe um modelo específico a ser seguido, basta usar uma folha de papel e um lápis ou um computador, ja que é um modelo pouco formal. Como o ideal é formar um diagrama visual, é recomendável usar textos curtos e objetivos para explicar uma atividade no desenho.

    2. Objetivo

    Um Rich Picture ajuda a equipe a alcançar um entendimento amplo sobre o tema e enriquece as discussões, o que faz dele um ótimo aliado nas reuniões de elicitação de requisitos de um projeto de software.
